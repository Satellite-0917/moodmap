## Idea 1: Zen Minimalist (젠 미니멀리스트)

### Design Movement
**Japandi (Japanese + Scandinavian)**: 일본의 와비사비(Wabi-sabi)와 북유럽의 기능주의가 결합된 스타일. 불완전함의 미학과 단순함의 조화를 추구합니다.

### Core Principles
1.  **여백의 미 (Ma)**: 비어 있는 공간을 통해 콘텐츠(사진)에 집중하게 합니다.
2.  **자연스러움 (Naturalness)**: 인위적인 장식을 배제하고 재질감과 부드러운 톤을 사용합니다.
3.  **정적임 (Stillness)**: 사용자의 마음을 차분하게 만드는 고요한 분위기를 조성합니다.
4.  **직관성 (Intuition)**: 설명 없이도 알 수 있는 물 흐르듯 자연스러운 인터랙션.

### Color Philosophy
따뜻하고 중립적인 톤을 사용하여 사진이 돋보이게 합니다.
*   **Background**: `oklch(0.98 0.01 90)` (따뜻한 크림 화이트) - 차가운 순백색 대신 눈이 편안한 색상.
*   **Text**: `oklch(0.3 0.02 30)` (짙은 차콜 그레이) - 검정보다 부드러운 대비.
*   **Accent**: `oklch(0.75 0.1 60)` (차분한 테라코타) - 자연스러운 포인트 컬러.

### Layout Paradigm
**비대칭 균형 (Asymmetrical Balance)**:
*   그리드는 정갈하지만, 전체적인 배치는 중앙 정렬을 피하고 좌우 비대칭을 활용하여 여백을 강조합니다.
*   타이틀과 설명은 작고 섬세하게 배치하여 사진 그리드가 압도적인 주인공이 되도록 합니다.

### Signature Elements
1.  **종이 질감 (Paper Texture)**: 배경에 아주 미세한 종이 질감을 추가하여 아날로그적인 감성을 더합니다.
2.  **얇은 선 (Fine Lines)**: 1px의 아주 얇은 선으로 구획을 나누어 깔끔함을 유지합니다.
3.  **부드러운 그림자 (Soft Shadows)**: 사진이 떠 있는 듯한 느낌을 주는 은은한 확산형 그림자.

### Interaction Philosophy
**느림의 미학 (Slow Interaction)**:
*   버튼 호버 시 급격한 색상 변화보다는 투명도나 크기가 서서히 변하는 부드러운 반응.
*   사진 드래그 앤 드롭 시 "착" 하고 달라붙는 듯한 햅틱 피드백 같은 시각적 효과.

### Animation
*   **Fade-in**: 요소들이 순차적으로 천천히 나타나는 페이드 인 효과.
*   **Scale**: 버튼 클릭 시 살짝 눌리는 듯한 미세한 스케일 변화.

### Typography System
*   **Headings**: 명조체 계열 (Serif) - 우아하고 차분한 느낌 (예: Noto Serif KR).
*   **Body**: 고딕체 계열 (Sans-serif) - 가독성을 높인 깔끔한 서체 (예: Pretendard).

## Idea 2: Soft Pop (소프트 팝)

### Design Movement
**Neumorphism + Soft UI**: 부드러운 곡선과 파스텔 톤을 활용하여 친근하고 말랑말랑한 느낌을 주는 스타일.

### Core Principles
1.  **친근함 (Friendliness)**: 둥근 모서리와 부드러운 색감으로 사용자에게 다가갑니다.
2.  **촉각적 (Tactile)**: 만지고 싶은 듯한 입체감과 볼륨감을 강조합니다.
3.  **즐거움 (Playfulness)**: 과하지 않은 귀여움과 위트를 담습니다.

### Color Philosophy
밝고 화사한 파스텔 톤으로 긍정적인 에너지를 전달합니다.
*   **Background**: `oklch(0.95 0.03 200)` (연한 하늘색) - 맑고 상쾌한 느낌.
*   **Text**: `oklch(0.25 0.05 260)` (딥 네이비) - 선명하지만 딱딱하지 않은 텍스트.
*   **Accent**: `oklch(0.85 0.15 80)` (소프트 오렌지) - 활기를 주는 포인트.

### Layout Paradigm
**카드형 레이아웃 (Card-based Layout)**:
*   모든 요소가 둥근 카드로 구성되어 있어 블록 장난감을 조립하는 듯한 느낌.
*   그리드 자체가 하나의 큰 놀이판처럼 보이도록 디자인.

### Signature Elements
1.  **슈퍼 라운드 (Super Rounded)**: 모든 모서리를 둥글게 처리하여 날카로움을 제거.
2.  **내부 그림자 (Inner Shadow)**: 버튼이나 입력창에 깊이감을 주는 내부 그림자 활용.
3.  **이모지 (Emoji)**: 텍스트 대신 적절한 이모지를 활용하여 감정을 전달.

### Interaction Philosophy
**통통 튀는 반응 (Bouncy Interaction)**:
*   버튼을 누르면 젤리처럼 튕기는 효과.
*   사진을 놓을 때 물방울이 튀듯 퍼지는 효과.

### Animation
*   **Bounce**: 요소 등장이나 상호작용 시 탄성 있는 움직임.
*   **Wiggle**: 드래그 중인 사진이 살짝 흔들리는 효과.

### Typography System
*   **Headings & Body**: 둥근 고딕체 (Rounded Sans) - 귀엽고 친근한 느낌 (예: Jua, Gowun Dodum).

## Idea 3: Cyber Brutalism (사이버 브루탈리즘)

### Design Movement
**Neo-Brutalism**: 원색적인 컬러, 굵은 선, 과감한 타이포그래피를 사용하여 날것의 느낌을 강조하는 스타일.

### Core Principles
1.  **대담함 (Boldness)**: 눈에 띄는 대비와 강렬한 표현.
2.  **솔직함 (Honesty)**: 꾸밈없는 구조와 기능을 그대로 드러냄.
3.  **레트로 퓨처 (Retro Future)**: 초기 웹의 감성을 현대적으로 재해석.

### Color Philosophy
고대비의 원색을 사용하여 시선을 사로잡습니다.
*   **Background**: `oklch(0.99 0 0)` (순수 화이트) 또는 `oklch(0.95 0.05 100)` (네온 옐로우).
*   **Text**: `oklch(0 0 0)` (리얼 블랙) - 가장 강한 대비.
*   **Accent**: `oklch(0.6 0.25 250)` (일렉트릭 블루) - 디지털적인 느낌.

### Layout Paradigm
**모듈러 그리드 (Modular Grid)**:
*   검은색 굵은 선으로 명확하게 구획된 그리드.
*   여백을 최소화하고 꽉 찬 느낌을 주는 레이아웃.

### Signature Elements
1.  **굵은 테두리 (Thick Borders)**: 모든 요소에 2px~4px의 검은 테두리 적용.
2.  **하드 섀도우 (Hard Shadows)**: 블러 없는 선명한 그림자로 입체감 표현.
3.  **모노스페이스 폰트 (Monospace Font)**: 코딩 화면 같은 느낌을 주는 등폭 글꼴.

### Interaction Philosophy
**즉각적인 반응 (Instant Interaction)**:
*   애니메이션을 최소화하고 즉각적으로 상태가 변하는 반응.
*   버튼 클릭 시 물리적인 스위치를 켜는 듯한 딱딱한 느낌.

### Animation
*   **Glitch**: 가끔씩 화면이 지직거리는 글리치 효과.
*   **Marquee**: 흐르는 텍스트로 뉴스 티커 같은 느낌 연출.

### Typography System
*   **Headings**: 굵은 산세리프 또는 디스플레이 폰트 (예: Anton, Impact).
*   **Body**: 모노스페이스 폰트 (예: JetBrains Mono, Noto Sans Mono).
